Project Mate CLI

高效的项目管理命令行工具

npm install -g project-mate-cli

核心特性

快速项目初始化

一键创建标准化的项目结构，快速开始开发

依赖管理

智能管理项目依赖，自动处理版本冲突

Git 操作自动化

简化 Git 工作流程，提高团队协作效率

配置管理

灵活的项目配置管理，支持多环境部署

安装指南

npm install -g project-mate-cli
npx project-mate-cli [command]

使用方法

pm init

初始化新项目

pm init my-project

pm config

管理项目配置

pm config set key value

pm git

Git 操作助手

pm git sync
