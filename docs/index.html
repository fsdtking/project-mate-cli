Project Mate CLI

高效的项目管理命令行工具

npm install -g project-mate-cli

核心特性

项目搜索

支持本地项目和 GitLab 项目的快速搜索，轻松定位所需项目

项目打开

快速搜索并打开本地项目，提高开发效率

脚本执行

交互式运行项目内的 npm 脚本，支持客户端和服务端项目的特殊标记

分支管理

为分支添加注释，方便追踪和查找，提升团队协作效率

配置管理

灵活的配置选项，支持本地项目路径和 GitLab 集成设置

安装指南

npm install -g project-mate-cli
npx project-mate-cli [command]

使用方法

项目打开

快速打开本地项目

pm open
pm open project-name

项目搜索

搜索本地或 GitLab 项目

pm search keyword
pm search keyword -g

脚本运行

交互式运行项目脚本

pm run

分支管理

管理分支注释

pm br set "分支说明"
pm br list

配置管理

查看和设置配置

pm config get
pm config set local-project-root-directory "/path/to/projects"
